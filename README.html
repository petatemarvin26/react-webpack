React TypeScript with Webpack Setup

Author

This project was written by Marvin Petate

Description

This boilerplate are powered by Webpack, Babel, ESlint and React library

Goals

This project aims to give understanding to our Web Developers how to setup React Application from scratch with organized development configuration and also for the production

This configuration was based on Webpack Documentation followed the standard and best practices that requires by Webpack, to have smooth development process to our developers and make them selves familliar with this pattern

Setup Development

Step 1: Install dependencies

npm install # node package manager yarn install # using yarn packacge manager

Step 2: Setup environment variables

touch .env.dev # .env.dev NODE_ENV='development' VERSION='1.0.1'
